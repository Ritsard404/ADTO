# How to use the compact ACE dashboard

## What this feature does

The compact ACE dashboard turns the ADMS workbook calendar view into a dense operational dashboard. It uses school records, academic terms, ACE sessions, sections, projects, facilitator assignments, and teacher data already stored in ADTO.

## When to use it

Use it when you need to quickly scan monthly implementation progress, session status, teacher participation, project activity, and school calendar coverage without opening Excel.

## Before you begin

Make sure sessions, school setup, projects, sections, and teacher assignments have been entered or imported.

## Steps

1. Open **Dashboard**.
2. Filter by school, month, grade, teacher, activity, or status.
3. Review the top summary strip for school, facilitator, sessions, hours, coders, projects, and active teachers.
4. Check the academic calendar overview for term dates and class-day counts.
5. Use the monthly calendar grid to scan session density and status by day.
6. Use the session table for detailed rows such as period, grade/section, subject, teacher, topic, delivery, completion, and remarks.

## What happens next

Metrics update from the selected school and month. Admins see all schools, facilitators see assigned schools, and school admins see membership-scoped school records.

## Tips or reminders

The dashboard is intentionally compact. Use filters to narrow the table if the month contains many sessions.

## Common questions or issues

If a month is empty, ADTO has no sessions for that school and month yet. Check the session workbench or workbook import status.
