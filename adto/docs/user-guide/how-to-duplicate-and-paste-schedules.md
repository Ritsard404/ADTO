# How to duplicate and paste ACE schedules

## What this feature does

The schedule tools help admins and ACE facilitators create repeated session schedules without retyping every row. You can preview rows copied from a previous week or day, paste rows from Excel, save reusable weekly templates, check warnings, and save only the rows that are ready.

## When to use it

Use this when a school follows a fixed weekly ACE schedule, when you need to copy last week's sessions into the next week, when you want to generate sessions from a recurring weekly slot, or when you already prepared session rows in Excel.

## Before you begin

Make sure the school, date, grade level, section, facilitator, and time are correct. Facilitators can only save schedules for assigned schools. Admins can manage schedules for any school and may override conflicts when needed.

## Steps

1. Open **ACE Sessions**.
2. Go to **Schedule Duplication and Excel Paste**.
3. To duplicate a schedule, choose the school, source date or week, target date or week, and optional filters.
4. Select how topics should be handled: copy exactly, clear for editing, or increment session-number text.
5. Click **Preview duplicate** and review the validation column.
6. Save only after the preview shows the rows you expect.
7. To paste from Excel, copy rows with columns such as Date, Day, Start Time, Grade Level, Section, Subject, Teacher, Facilitator, Venue, Activity Type, Topic, and Remarks.
8. Paste the rows, click **Preview pasted rows**, review warnings, then save.
9. To use a recurring schedule, fill in **Recurring Schedule Template** with the school, weekly day, time, class, facilitator, venue, activity, and topic.
10. Click **Save template**.
11. Select the template, choose a start and end date, add any excluded no-class dates, then click **Preview template**.
12. Review the generated rows and save them only when the preview is correct.

## What happens next

Saved rows become ACE session records with a scheduled status. Completion-specific fields such as actual date, completion status, evidence, and final remarks are not copied from the source session. Template-generated sessions use the same preview and conflict checks as duplicated or pasted sessions.

## Tips or reminders

Use the preview warnings to catch duplicate sessions, facilitator conflicts, section conflicts, or venue conflicts before saving. Facilitators should ask an admin before saving rows with major conflicts.

Put holidays, school events, and no-class days in the excluded dates field before previewing a template range.

## Common questions or issues

If no rows appear in the duplicate preview, check that the source week or day has sessions for the selected school and filters. If pasted facilitator names do not match system users, ADTO uses the active user unless a valid facilitator is selected or recognized.
