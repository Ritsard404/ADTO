# How to manage facilitator assignments

## What this feature does

Facilitator assignments track the current school for each ACE facilitator while preserving every previous school assignment.

## When to use it

Use it when assigning a facilitator to a school, transferring a facilitator to another school, closing a completed assignment, or reviewing school implementation continuity.

## Before you begin

Only admins can assign or transfer facilitators. Confirm the facilitator and destination school before saving an active assignment.

## Steps

1. Open **Facilitators**.
2. Choose the school, facilitator, start date, and status.
3. Select **ACTIVE** when the facilitator is being assigned to the current primary school.
4. Save the assignment.
5. Review **Current and Historical Assignments** to confirm the previous active assignment is now marked **TRANSFERRED**.
6. Use **Complete** only when the facilitator has finished the assignment without being moved to another school.

## What happens next

ADTO keeps the old assignment, sessions, reports, projects, evidence, inventory records, and analytics linked to the original school. The facilitator only receives write access to the active assignment.

## Tips or reminders

A facilitator can have only one active primary school assignment at a time. School reports still show all facilitators involved from assignment history, even if they are now assigned elsewhere.

## Common questions or issues

If a facilitator cannot update a school, check that the assignment is still **ACTIVE**. Transferred or completed assignments remain visible for historical reporting but are locked for normal facilitator updates.
