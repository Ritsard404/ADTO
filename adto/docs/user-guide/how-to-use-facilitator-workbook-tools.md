# How to use facilitator workbook tools

## What this feature does

Facilitator workbook tools bring the key ADMS workbook views into ADTO: monthly QuickView, school setup details, teacher participation, project register fields, inventory verification, evidence links, and report preview.

## When to use it

Use these tools when checking monthly implementation progress, preparing reports, reviewing student projects, verifying ACE kit readiness, or linking evidence from Drive.

## Before you begin

You must have an active facilitator assignment for the school. If a school is missing, ask an admin to check your assignment history.

## Steps

1. Open **Facilitator Workspace** to review the Monthly QuickView.
2. Filter by school and month to compare this-month totals with cumulative totals.
3. Open **Facilitator Schools** to review adoption setup, term dates, sections, and teacher participation.
4. Open **Projects** to update project title, term, students involved, grade/section, teacher, type, description, link, remarks, status, and submission date.
5. Open **Inventory** to verify issued and total quantity, borrowed status, condition, completeness, remarks, and sign-off.
6. Open **Evidence** to add Drive or external evidence links for a school, session, or project.
7. Open **Reports** to preview school metrics and submit monthly narratives, challenges, recommendations, school updates, and quick insights.

## What happens next

Updated data appears in the facilitator workspace, admin dashboard, report previews, and official report inputs.

## Tips or reminders

Use clear evidence names and links. Keep project links clickable and inventory remarks specific enough for admin follow-up.

## Common questions or issues

If an empty month shows zero sessions or projects, it means ADTO has no records for that school and month yet. Check the schedule, import status, or project register.
