# How to use the AI report assistant

## What this feature does

The AI report assistant reads the selected report preview metrics and gives report-ready guidance, missing-data checks, and next actions for admins, school admins, and facilitators.

## When to use it

Use it before generating a PPT/PDF report, when writing monthly monitoring narratives, or when checking why a report looks incomplete.

## Before you begin

Open **Reports** and select the school, school year, and report type. The assistant only uses the report metrics visible to your account.

## Steps

1. Open **Reports**.
2. Preview the report for the school and school year.
3. Use **AI Report Assistant** to ask for a report draft, data gaps, or next actions.
4. Review the answer against the visible metrics before copying it into a report narrative.

## What happens next

If an OpenAI API key is configured on the server, ADTO uses it for the answer. If not, ADTO still returns local guidance from the report metrics.

## Tips or reminders

Do not use the assistant to invent counts, project evidence, or attendance details. Fix missing sessions, project links, teacher data, or inventory remarks in the matching module first.
