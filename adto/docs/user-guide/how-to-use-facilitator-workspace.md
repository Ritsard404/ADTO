# How to use the Facilitator Workspace

## What this feature does

The Facilitator Workspace gathers assigned schools, ACE sessions, schedules, projects, inventory checks, evidence, reports, and analytics in one role-specific area.

## When to use it

Use it when you are preparing school schedules, logging session completion, checking project outputs, updating ACE hardware status, uploading evidence, or preparing monitoring reports.

## Before you begin

Make sure your account is assigned to the correct school. Facilitators can only see assigned schools.

## Steps

1. Open **Facilitator Workspace** from the sidebar.
2. Review the dashboard cards for upcoming sessions, completed sessions, coding hours, reports, and inventory alerts.
3. Use **Schedules** to create, duplicate, repeat, or paste session schedules from Excel.
4. Use **Schools** to review assigned school setup, sections, teachers, adoption type, and schedule arrangement.
5. Use **Projects** to monitor computational artifacts and completion status.
6. Use **Inventory** to verify quantities, condition, missing items, damaged items, and return notes.
7. Use **Evidence** to review session, project, attendance, consultation, and school activity files.
8. Use **Reports** and **Analytics** to prepare monthly monitoring and year-end metrics.

## What happens next

ADTO keeps facilitator records scoped to the school where they were created and uses the same fields from the ACE workbook: grade and section, subject, teacher, activity, delivery, completion, remarks, project documentation, and inventory condition. Previous assignments remain visible for history but are locked for normal updates.

## Tips or reminders

Use schedule duplication and Excel paste for repeated weekly schedules. Keep remarks clear when sessions are cancelled, rescheduled, or completed.

## Common questions or issues

If a school is missing, ask an admin to check your active facilitator assignment. If an inventory item needs repair or is missing, update the condition and remarks before generating reports.
